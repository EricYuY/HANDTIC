Para ejecutar la simulacion, se deben seguir los siguientes pasos:
1. Crear una carpeta catkin_ws para almacenar los archivos de ROS
2. Instalar los paquetes de turtlebot3
3. Compilar el paquete usando catkin_make
4. Crear un nuevo paquete usando catkin_create_package
5. Insertar el folder presentado en una carpeta llamada "src"
6. Compilar los paquetes usando catkin_make
